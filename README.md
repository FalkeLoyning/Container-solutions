# Container Solutions

Mock frontend for testing container management ideas – no backend required.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## What's Included

- Dashboard with container metrics
- Container list with start/stop/remove actions
- Create new containers via modal dialog
- All data is mock – no backend needed

## Tech Stack

- **Next.js 15** – React framework with App Router
- **React 19** – UI components
